Getting started

Installation

Assuming node.js is already installed, install mineral.js by using the following command 


          $ npm install mineraljs --save

It's as easy as that.

Configuration

ServerOptions

Key Value Description
controllers Function[] 
                  import { UserController, AddressController } from './controllers' 

                  new Server({
                    controllers: [UserController, AddressController]
                  })
components Function[] 
                  import { UserService, AddressService } from './services' 

                  new Server({
                    components: [UserService, AddressService]
                  })
Entities Function[] 
                  import { User, Address } from './entities' 

                  new Server({
                    entities: [User, Address]
                  })
logLevel Enum Possible values: DEBUG, INFO, WARNING, ERROR
default value: ERROR
amqpUrl string if set, amqp will be enabled
dbOptions DbOptions see DbOptions

DbOptions

Key Value Description
host string host to the database.
example: 'localhost'
type string type of the database.
example: 'postgres'
port number port of the database.
example: 5432
username string username of the database.
example: 'postgres'
password string password of the database.
example: 'postgres'
database string database name.
example: 'mineraljs'
logging boolean if true, logging of the queries will be enabled.
default value: If loglevel is on DEBUG, true, otherwise, false
